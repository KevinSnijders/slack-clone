## Installation

- Use "create-react-app"
- Configure your Firebase settings
- Run the code

### Clone

- Clone this repo to your local machine using `https://github.com/SOMEUSER/SOMEREPO`

### Setup

> Install npm packages

```shell
$ npm install
```

> Run application

```shell
$ npm start
```
